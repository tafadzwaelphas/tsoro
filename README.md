# Tsoro

A 4-row Shona mancala game built with plain HTML and JavaScript.

*Tsoro* is the generic Shona word for board games; this is the 4-row sowing (mancala) variant, not to be confused with **Tsoro Yematatu**, a different 3-piece alignment game from the same tradition.

Play it live: https://tafadzwaelphas.github.io/tsoro/

## Features

- Single-player vs AI opponent
- Animated seed movement
- Undo support
- Responsive design
